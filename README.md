# Homebrew Tap for Echo

Privacy-first voice-to-text for macOS. All processing happens on-device.

## Install

```bash
brew tap apopen/echo
brew install --cask echo
```

## Update

```bash
brew upgrade --cask echo
```

## Uninstall

```bash
brew uninstall --cask echo
```
